This project demonstrates audio playback on the
PIC32CM5164LE00100 microcontroller using the DAC peripheral.

The project includes multiple audio outputs such as alarm
tones and a custom recorded voice, intended for alert and
notification applications.

## Features
- Two different alarm audio signals
- One custom recorded voice audio
- DAC-based audio output
- Timer-controlled audio sampling
- Software-based audio selection

## Hardware
- MCU: PIC32CM5164LE00100
- Audio Output: DAC
- Speaker / Amplifier (external)

## Tools
- MPLAB X IDE
- XC32 Compiler

## Applications
- Industrial alert systems
- Safety and warning systems
- Embedded audio notifications

## Notes
- Audio files are stored in RAW format
- Audio samples are played using DAC with timer control
